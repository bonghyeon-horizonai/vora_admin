# 상품 관리 기능 요구사항 (Admin Product Management)

이 문서는 '어드민_상품 목록' 화면의 디자인을 바탕으로 상품 관리 시스템의 기능 요구사항을 정리합니다.

## 1. 페이지 개요
- **목적**: 서비스에서 판매하는 구독 상품 및 개별 상품을 관리합니다.
- **진입 경로**: 사이드 메뉴 -> 상품 관리 -> **상품 목록**

## 2. 상세 기능 요구사항

### 2.1 상품 목록 조회
- **목록 표시**: 등록된 전체 상품 목록을 테이블 형태로 표시합니다.
- **데이터 필드**:
  - **상품명**: 서비스에서 표시되는 상품의 명칭 (예: PPT 생성, PPT 요약)
  - **가격 (USD)**: 상품의 판매 가격 (미화 달러 기준)
  - **주기**: 결제 또는 구독 주기 (예: 월간)
  - **상태**: 상품의 판매 및 활성화 상태 (예: 활성)
  - **작성자**: 상품을 등록한 관리자명
  - **작성일**: 상품이 시스템에 등록된 날짜 및 시간
  - **수정일**: 상품 정보가 마지막으로 수정된 날짜 및 시간 (수정 이력이 없는 경우 '-' 표시)
  - **작업**: 해당 상품을 수정하거나 삭제할 수 있는 액션 버튼

### 2.2 검색 및 필터링
- **통합 검색**: 상품명 또는 상세 내용을 키워드로 검색할 수 있습니다.
  - 검색어 입력 후 '엔터(Enter)' 키를 누르면 검색이 수행됩니다.
- **상태 필터**: 상품의 판매 상태를 기준으로 목록을 필터링할 수 있는 드롭다운 필터를 제공합니다.

### 2.3 상품 등록 및 상세 관리
새로운 상품을 등록하거나 기존 상품의 상세 정보를 관리하는 기능으로, 단계별 입력(탭) 방식과 도구 번들링(도구 선택) 기능을 포함합니다.

#### 2.3.1 상품 속성 설정
- **구분/주기**: 상품의 유형(일반 등)과 과금 주기(월간 등)를 선택합니다.
- **카테고리**: 상품이 속한 서비스 카테고리를 드롭다운에서 선택합니다.
- **1회성 여부**: 해당 상품이 1회성 구매 상품인지 여부를 설정합니다.
- **상품 코드**: 시스템 고유 코드를 확인하고 복사할 수 있으며, 필요 시 **Paddle 연동** 기능을 통해 외부 결제 시스템과 연결합니다.

#### 2.3.2 다국어 가격 정보 입력
- **상품명 및 가격**: 한국어(KRW), 영어(USD), 일본어(JPY) 각각의 상품 명칭과 통화별 가격을 입력할 수 있는 필드를 제공합니다.

#### 2.3.3 도구/제공량 선택 (도구 번들링)
- **도구 검색 및 선택**: 서비스에서 제공하는 도구 목록에서 상품에 포함할 도구를 검색하여 선택합니다.
- **선택된 도구 리스트 관리**:
  - **순서 변경**: 드래그 앤 드롭 아이콘을 사용하여 도구 노출 순서를 변경할 수 있습니다.
  - **월 제공량 설정**: 각 도구별로 한 달 동안 제공할 사용량을 입력합니다.
  - **삭제**: 선택된 리스트에서 도구를 제외할 수 있는 삭제 버튼을 제공합니다.

#### 2.3.4 작업 제어
- **임시 저장**: '다음' 버튼 클릭 시 현재 입력 내용이 자동으로 임시 저장됨을 안내합니다.
- **취소 및 다음**: 등록 절차를 취소하거나 다음 단계(상품 상세)로 이동할 수 있는 버튼을 제공합니다.

### 2.4 상품 상세 정보 관리 (상세 탭)
'상품 상세' 탭에서는 상품의 다국어 설명과 시각적 에셋, 최종 활성화 상태를 관리합니다.

#### 2.4.1 다국어 상품 설명
- **한국어/영어 설명**: 상품의 특징과 혜택을 상세히 설명할 수 있는 다국어 텍스트 영역(Textarea)을 제공합니다.

#### 2.4.2 상품 아이콘 이미지 관리
- **이미지 업로드**: 권장 사이즈(512x512px)에 맞춘 상품 아이콘을 업로드하거나 변경할 수 있는 기능을 제공합니다.
- **경로 관리**: 업로드된 이미지의 파일 경로 또는 URL을 확인할 수 있는 필드를 제공합니다.

#### 2.4.3 최종 활성화 및 저장
- **활성화 토글**: 해당 상품을 사용자에게 노출할지 여부를 결정하는 '활성화' 토글 스위치를 제공합니다.
- **이전 및 저장**: 이전 단계(기본 정보)로 돌아가거나, 모든 설정을 완료하고 시스템에 반영하는 `[저장]` 버튼을 제공합니다.
