# 도구 관리 기능 요구사항 (Admin Tool Management)

이 문서는 '어드민_도구 목록' 화면의 디자인을 바탕으로 도구 관리 시스템의 기능 요구사항을 정리합니다.

## 1. 페이지 개요
- **목적**: 서비스에서 제공하는 다양한 AI 도구(PPT 생성, PDF 요약 등)를 관리하고 워크플로우와 연동합니다.
- **진입 경로**: 사이드 메뉴 -> 상품 관리 -> **도구 목록**

## 2. 상세 기능 요구사항

### 2.1 도구 목록 조회
- **목록 표시**: 등록된 전체 도구 목록을 테이블 형태로 표시합니다.
- **데이터 필드**:
  - **도구명**: 사용자가 식별할 수 있는 도구의 이름 (예: PPT 생성, PDF 요약)
  - **도구 코드**: 시스템 내부에서 사용하는 고유 코드 (예: `tool_abc123`)
  - **워크플로우 연동**: 해당 도구와 연결된 백엔드 워크플로우 식별자 (예: `vora_ppt_generator`)
  - **상태**: 도구의 활성화 상태 (예: 활성, 비활성)
  - **작성자**: 해당 도구를 등록하거나 수정한 관리자명
  - **등록일**: 도구가 시스템에 최초 등록된 날짜 및 시간
  - **작업**: 해당 행의 도구를 수정하거나 삭제할 수 있는 액션 버튼

### 2.2 검색 및 필터링
- **통합 검색**: 도구명 또는 도구 내용(설명 등)을 키워드로 검색할 수 있습니다.
  - 검색어 입력 후 '엔터(Enter)' 키를 누르면 검색이 수행됩니다.
- **상태 필터**: 도구의 상태(전체, 활성, 비활성 등)를 기준으로 목록을 필터링할 수 있는 드롭다운을 제공합니다.

### 2.3 도구 등록 및 관리
- **도구 등록**: 화면 우측 상단의 `[+ 도구 등록]` 버튼을 통해 새로운 도구를 시스템에 추가할 수 있는 폼으로 이동합니다.
- **상세 정보 수정**: 목록의 '작업' 컬럼에 있는 수정(연필) 아이콘을 클릭하여 기존 도구의 정보를 수정할 수 있습니다.
- **도구 삭제**: 목록의 '작업' 컬럼에 있는 삭제(쓰레기통) 아이콘을 클릭하여 도구를 시스템에서 제거할 수 있습니다. (삭제 전 확인 팝업 필요)

### 2.4 도구 세부 정보 수정
목록에서 수정 아이콘을 클릭하여 진입하며, 도구의 명칭, 속성, 연동 정보 등을 상세히 관리합니다.

#### 2.4.1 도구 기본 정보 (다국어 지원)
- **도구명**: 한국어, 영어, 일본어 각각의 도구명을 입력할 수 있는 입력 필드를 제공합니다.
- **도구 설명**: 한국어, 영어 등 언어별로 도구에 대한 상세 설명을 입력할 수 있는 다국어 텍스트 영역(Textarea)을 제공합니다.

#### 2.4.2 도구 속성 관리
- **구분**: 도구가 유료인지 무료인지 선택할 수 있는 드롭다운 메뉴를 제공합니다.
- **Tier**: 도구의 서비스 등급(예: 1, 2, 3 등)을 선택할 수 있는 드롭다운 메뉴를 제공합니다.

#### 2.4.3 도구 연동 정보
- **워크플로우 연동**: 백엔드 워크플로우 식별자를 입력하거나 선택할 수 있는 필드를 제공합니다.
- **도구 코드**: 시스템 내부 고유 코드를 확인하고 필요한 경우 수정하거나 생성할 수 있는 필드를 제공합니다. (복사 기능 포함)

#### 2.4.4 도구 아이콘 이미지
- **이미지 업로드**: 권장 사이즈(512x512px)에 맞춘 도구 아이콘 이미지를 업로드하거나 변경할 수 있는 기능을 제공합니다.
- **파일 링크**: 업로드된 이미지의 경로 또는 링크를 관리합니다.

#### 2.4.5 상태 및 저장
- **활성화 토글**: 도구의 사용 여부를 직관적으로 전환할 수 있는 '활성화' 토글 스위치를 제공합니다.
- **저장**: 입력한 정보를 시스템에 반영하는 `[저장]` 버튼을 제공합니다.

### 2.5 워크플로우 연동 기능
- 등록된 도구는 각각 고유한 백엔드 워크플로우 아이디와 매핑되어야 합니다.
- 매핑된 워크플로우를 통해 실제 AI 서비스 로직이 수행됩니다.
